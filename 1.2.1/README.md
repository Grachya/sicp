##1.1.2 Древовидная рекурсия

(define (fib n)
  (cond ((= n 0) 0)
        ((= n 1) 1)
        (else (+ (fib (- n 1))
                  (fib (- n 2))))))

Древовидная рекурсия растет (число шагов) экспоненциально при увеличении n.
Это число ближайшее к f^2/√5 
Требования к памяти растут линейно.

b <- a - число b это предыдущее число
